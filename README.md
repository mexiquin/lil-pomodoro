# Pomodoro Timer

A little pomodoro timer I use for getting stuff done

## Usage

Open `index.html` in any modern browser. Or use the GitHub page version :)

### Controls

| Button | Description |
|--------|-------------|
| Start | Begin the current phase |
| ⏸ | Pause the timer |
| ⏭ | Skip to the next phase |
| Restart | Reset the current phase |
| New Session | Start over from session 1 |


No dependencies - single HTML file with embedded CSS and JavaScript. Uses Web Audio API for sound generation.
